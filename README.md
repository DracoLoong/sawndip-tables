# 方块壮字单字码表

## 简介

本项目提供一份方块壮字的单字码表，旨在整理、规范并提升现有方块壮字编码资料的可用性与一致性。

## 编制说明

1. 目前网络上流传的方块壮字码表，普遍混杂大量Unicode 私用区（PUA）编码，在检索、显示和跨平台使用上存在诸多不便。
   本码表对相关内容进行了整理，将原有私用区编码迁移至 Unicode 已正式分配的编码位，并*删除未获得编码的条目，以提高整体可用性和规范性。

2. 本码表的主要收录依据为《古壮字字典·初稿》。
   对于尚未编码的壮字，一律不予收录。
   同时，兼收部分来源于《古壮字大字*》以及 *Mapping the Old Zhuang Character Script* 的条目，并参考了部分字统网的数据进行校对与补充。

3. 本码表仅收录单字条目，不包含合文、异体合写或未能确定编码归属的字符。

## 后续计划

接下来的工作计划包括但不限于：

* 对现有条目进行校对，修正错误，并增补尚未收录但已有编码的方块壮字
* 加入汉字的老借词音（平话音）读音信息
* 加入汉字新借词音（官话音）的类推读音条目

## 说明

本项目仍在持续整理与完善中，码表内容可能随 Unicode 编码进展及文献校勘结果而调整。
欢迎对方块壮字、古壮文或相关编码问题感兴趣的研究者与使用者提出建议或指正。
